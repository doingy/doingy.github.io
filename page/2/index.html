ERROR ITMS 90542

发表于 2016-01-13 | | 阅读次数:

打包后，先不要[Upload to App Store…]上传

在要上传的包上右键[Show in Finder]然后依次展开[xxxx.xcarchive][Products][Applications][xxxx] // xxxx–项目名
然后搜索所有的info.plist
找到是哪个plist含有[CFBundleSupportedPlatforms]属性，看是那个三方库包含的这个plist，然后到Xcode里面删除该属性。

ERROR ITMS 90096

发表于 2016-01-13 | | 阅读次数:

解决办法在Assets.xcassets提供启动图片

因为我们的App只有横屏版本，最开始的时候没有勾选iPhone,所以一直报错。
iPhone在iOS8.0以前只有Portrait,没有Landscape，所有启动图片的规则必须是竖版的
即
Default@2x.png [640x960]
Default-568h@2x.png [640x1136]
