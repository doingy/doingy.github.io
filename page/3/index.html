MacBook不能识别外接键盘

发表于 2015-10-31 | | 阅读次数:

最近把系统升级到10.11之后，出现不能识别外接键盘的问题。
使用了如下几种方法解决问题，还给苹果客服打了求助电话
1.重新安装系统，安装之后，没问题了。但是中途拔了一次键盘，omg，又不能识别了
顺便吐槽下，最近从mac Appstore下载软件必定出现『下载失败，请到已购再次下载』的提示，O^O
2.重置RPM什么之类的，总之是关机按Option+Control+R+P，等等，但是，没有效果

最后，尝试在键盘和电脑中间接了个usb hub，外接键盘神奇的好用了。=_=!
反正我这里是好用了，有需求的可以试试
